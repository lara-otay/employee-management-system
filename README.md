# EmployeeManagement

Simple ASP.NET Core MVC Employee Management demo using EF Core InMemory provider.

Open: https://localhost:5001/ or http://localhost:5000/ then navigate to /Employees
Running tests
1. From workspace root run:

   dotnet test EmployeeManagement.Tests\EmployeeManagement.Tests.csproj

This will build and run the xUnit tests for the EmployeeService using the EF InMemory provider.
